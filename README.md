# 📊 Sales Data Dashboard (Excel)

## 📌 Overview
An interactive **Excel Sales Dashboard** built using Pivot Tables, Charts, and Slicers to analyze sales performance, profit trends, customer contributions, and category-wise insights.

## 📊 Key Metrics
- **Total Sales:** $1,928,887.86
- **Total Profit:** $247,961.52

## ⚙️ Features
- KPI Cards for Sales & Profit
- Category-wise Profit Analysis
- Customer-wise Profit Analysis
- Monthly Sales Trend
- Interactive Filters (Month, Year, Category)

## 🛠 Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers
